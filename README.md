# ✨ Features

## 📅 Period Management

* Create weekly or custom financial periods
* Organize expenses by date ranges
* View period summaries instantly

## 💰 Expense Tracking

* Add daily expenses
* Categorize spending
* Track income and expenses separately
* Real-time expense calculations

## 📈 Analytics & Breakdown

* Daily expense breakdown
* Category-based analysis
* Spending summaries
* Balance calculations

## 🔔 Notifications

* Instant notifications after adding expenses
* Local notification support
* Smooth user feedback experience

# 🚀 Getting Started

## Clone the repository

```bash
git clone https://github.com/your-username/fintrack.git
```

## Open the project

```bash
cd fintrack
```

## Install dependencies

```bash
flutter pub get
```

## Run the app

```bash
flutter run
```
